The most dreamed fireworks and balloons for the most your dream moments

The website was created with Wordpress CMS while the e-shop functionality was implemented with the Woocommerce plugin
The content is written in Greek but there is a plugin ready to use which will add multilingual function in the final version of the website
There is also a plugin which connects the bank of the store in the e-shop in order to accept payment by debit / credit cards
Also, for the needs of the project, I developed custom functions in PHP that override the default image of each product with the respective video from Youtube
