24/7 Music Webradio

This mini-site was created with simple HTMl and CSS
There is one HTML5 player that is supported on all systems (computers, tablets, smartphones)
The webradio is based on Icecast Server and its management is done with Centova Panel
It is configured to transmit its own music with specific automation conditions
This configuration is supported by the auto-dj function which allows us to upload our own music
