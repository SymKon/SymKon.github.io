Audio File Evaluation System

This evaluation system was constructed according to the specifications and requirements set by the European School Radio
The implementation was done with php without any framework or other platform and includes:

  • School Assessment System (access only by teachers)
  • Simple listening to audio by students with a special code per school (which is displayed to teachers)
  • Judge Rating System
  • Score for calculating and displaying results (access only by organizing committee)
For the implementation it was necessary to interface with the existing site (Wordpress) for the provision of users as well as with the existing custom web application (CakePHP) for access to audio and additional details.
