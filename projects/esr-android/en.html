Webradio Android App

The application was created for Android devices in Android Studio in Java
Includes:

  • Screen for listening to the radio and chatting
  • Screen that shows the next 4 shows that will be broadcast
  • Screen displaying the program (by selecting a day from the calendar)
  • The broadcast episodes that have already been broadcast
  • The latest radio news
  • Settings screen
The application was implemented in a bilingual environment (Greek, English) with the ability for the user to change the language at any time.
